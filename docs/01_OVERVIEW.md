## 一、项目简介

**Paste Library** 是一款现代化的剪贴板管理工具，实时监听系统剪贴板，保存历史记录，支持快速查询和管理。

**核心特性**:
- 🔴 实时剪贴板监听（文本/HTML/RTF/图片/文件/文件夹/多文件）
- 🔴 SQLite 数据持久化，本地存储
- 🔴 智能搜索（模糊匹配 + 拼音 + 语法）
- 🔴 多设备局域网同步（零成本）
- 🔴 跨平台（Windows/macOS/Linux + iOS/Android）

---

## 二、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端框架** | Vue 3 + TypeScript | Composition API + `<script setup>` |
| **构建工具** | Vite | 快速开发体验 |
| **桌面框架** | Tauri v2 | 跨平台桌面应用 |
| **后端语言** | Rust | 高性能系统编程 |
| **数据库** | SQLite | 本地数据持久化 |
| **移动端** | Tauri Mobile / Native | iOS/Android 支持 |

### 核心插件
- `tauri-plugin-clipboard-x` - 剪贴板监听
- `tauri-plugin-global-shortcut` - 全局快捷键
- `tauri-plugin-tray` - 系统托盘
- `tauri-plugin-autostart` - 开机自启
