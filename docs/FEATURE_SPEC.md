# Paste Library - 功能规格说明

## 设置面板功能规格

### 1. 剪贴板设置

**窗口设置**
- 窗口位置: remember/center/cursor
- 激活时回到顶部: boolean
- 激活时切换至全部分组: boolean

**音效设置**
- 复制音效: boolean

**搜索设置**
- 搜索框位置: top/bottom
- 默认聚焦: boolean
- 自动清除: boolean

**内容设置**
- 自动粘贴: off/single/double
- 图片OCR: boolean
- 复制为纯文本: boolean
- 粘贴为纯文本: boolean
- 删除确认: boolean
- 自动排序: boolean

### 2. 历史记录设置

- 保留时长: number (0=永久)
- 最大保留条数: number (0=无限)
- 删除历史记录: 危险操作按钮

### 3. 通用设置

**应用设置**
- 登录时启动: boolean
- 静默启动: boolean
- 显示菜单栏图标: boolean
- 显示任务栏图标: boolean

**外观设置**
- 界面语言: zh-CN/en-US
- 主题模式: system/light/dark

**应用更新**
- 自动检查更新: boolean
- 参与测试版体验: boolean

### 4. 快捷键设置

- 打开剪贴板窗口: string (显示)
- 打开偏好设置窗口: string (可配置)
- 快速粘贴: boolean + modifier
- 粘贴为纯文本: string

### 5. 数据备份

**存储设置**
- 数据存储路径: string (显示+复制+打开)
- 日志存储路径: string (显示)

**导入和导出**
- 导入数据: 按钮
- 导出数据: 按钮

### 6. 关于

- 应用信息
- 打开剪贴板按钮

## 剪贴板内容类型规格

### 支持的内容类型

| 类型 | 存储方式 | 显示方式 |
|------|----------|----------|
| text | 数据库存储 | 文本预览 |
| html | 数据库存储 | 纯文本预览 |
| rtf | 数据库存储 | 纯文本预览 |
| image | 文件系统+路径 | 缩略图+尺寸 |
| file | 路径存储 | 图标+文件名+大小 |
| folder | 路径存储 | 图标+名称+项目数 |
| files | 路径列表JSON | 摘要 |

## 交互规格

### 鼠标操作

| 操作 | 行为 | 可配置 |
|------|------|--------|
| 左键单击 | 复制/粘贴 | ✅ |
| 双击 | 粘贴 | ❌ |
| 右键 | 上下文菜单 | ❌ |

### 上下文菜单项

- 复制 (Ctrl+C)
- 粘贴 (Ctrl+V)
- 复制为纯文本
- 打开文件
- 在文件夹中显示
- 收藏/取消收藏
- 删除

### 菜单项可见性

| 菜单项 | text | image | file | folder | files |
|--------|------|-------|------|--------|-------|
| 复制 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 粘贴 | ✅ | ✅ | ✅ | ❌ | ❌ |
| 复制为纯文本 | ✅ | ❌ | ❌ | ❌ | ❌ |
| 打开文件 | ❌ | ✅ | ✅ | ❌ | ❌ |
| 在文件夹中显示 | ❌ | ✅ | ✅ | ✅ | ❌ |
| 收藏 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 删除 | ✅ | ✅ | ✅ | ✅ | ✅ |
