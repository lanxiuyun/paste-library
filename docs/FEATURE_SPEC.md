# Paste Library - 功能规格说明

## 设置面板功能规格

### 1. 剪贴板设置

**窗口设置**
- 窗口位置: remember/center/cursor
- 激活时回到顶部: boolean
- 激活时切换至全部分组: boolean

**音效设置**
- 复制音效: boolean

**搜索设置**
- 搜索框位置: top/bottom
- 默认聚焦: boolean
- 自动清除: boolean

**内容设置**
- 自动粘贴: off/single/double
- 图片OCR: boolean
- 复制为纯文本: boolean
- 粘贴为纯文本: boolean
- 删除确认: boolean
- 自动排序: boolean

### 2. 历史记录设置

- 保留时长: number (0=永久)
- 最大保留条数: number (0=无限)
- 删除历史记录: 危险操作按钮

### 3. 通用设置

**应用设置**
- 登录时启动: boolean
- 静默启动: boolean
- 显示菜单栏图标: boolean
- 显示任务栏图标: boolean

**外观设置**
- 界面语言: zh-CN/en-US
- 主题模式: system/light/dark

**应用更新**
- 自动检查更新: boolean
- 参与测试版体验: boolean

### 4. 快捷键设置

- 打开剪贴板窗口: 按键录制按钮 (实时捕获组合键)
  - 支持: Ctrl, Alt, Shift, Win 修饰键
  - 支持: 字母键 A-Z, 数字键 0-9, F键 F1-F12
  - 支持: 特殊键 Space, Enter, Escape, Tab, Backspace, Delete, 方向键
  - 修改后需重启应用生效
- 打开偏好设置窗口: string (可配置)
- 快速粘贴: boolean + modifier
- 粘贴为纯文本: string

### 5. 数据备份

**存储设置**
- 数据存储路径: string (显示+复制+打开)
- 日志存储路径: string (显示)

**导入和导出**
- 导入数据: 按钮
- 导出数据: 按钮

### 6. 关于

- 应用信息
- 打开剪贴板按钮

## 剪贴板内容类型规格

### 支持的内容类型

| 类型 | 存储方式 | 显示方式 |
|------|----------|----------|
| text | 数据库存储 | 文本预览（最多3行） |
| html | 数据库存储 | 纯文本预览（最多3行） |
| rtf | 数据库存储 | 纯文本预览（最多3行） |
| image | 文件系统+路径 | 缩略图+尺寸 |
| file | 路径存储 | 图标+文件路径（图片文件显示预览） |
| folder | 路径存储 | 图标+文件夹路径 |
| files | 路径列表JSON | 前3个文件路径列表 |

## 交互规格

### 鼠标操作

| 操作 | 行为 | 可配置 |
|------|------|--------|
| 左键单击 | 复制/粘贴 | ✅ 通过 `click_action` 设置 |
| 双击 | 复制/粘贴 | ✅ 通过 `double_click_action` 设置 |
| 右键 | 上下文菜单 | ❌ |

### 上下文菜单项

- 复制 (Ctrl+C) - 仅复制到剪贴板
- 粘贴 (Ctrl+V) - 复制并执行粘贴（隐藏窗口 + 模拟快捷键）
- 添加到队列
- 复制为纯文本
- 打开文件
- 在文件夹中显示
- 复制文件路径
- 收藏/取消收藏
- 删除

### 菜单项可见性

| 菜单项 | text | image | file | folder | files |
|--------|------|-------|------|--------|-------|
| 复制 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 粘贴 | ✅ | ✅ | ✅ | ❌ | ❌ |
| 添加到队列 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 复制为纯文本 | ✅ | ❌ | ❌ | ❌ | ❌ |
| 打开文件 | ❌ | ✅ | ✅ | ❌ | ❌ |
| 在文件夹中显示 | ❌ | ✅ | ✅ | ✅ | ❌ |
| 复制文件路径 | ❌ | ❌ | ✅ | ✅ | ✅ |
| 收藏 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 删除 | ✅ | ✅ | ✅ | ✅ | ✅ |

## 详情窗口规格

### 类型特定视图

**文本类型**
- 可编辑的文本区域
- 编辑/预览模式切换
- 底部统计栏：字符数、单词数、行数、大小
- 操作按钮：复制、粘贴、保存为新项

**图片类型**
- 大图预览
- 图片信息：尺寸、格式、大小
- 操作按钮：复制、粘贴

**文件类型**
- 文件图标
- 文件信息：名称、大小、完整路径
- 操作按钮：复制、粘贴、复制路径

**文件夹类型**
- 文件夹图标
- 文件夹信息：名称、路径
- 操作按钮：复制、粘贴、复制路径

**多文件类型**
- 文件数量统计
- 文件列表（显示所有文件路径）
- 每个文件显示对应图标（图片/文件）
- 操作按钮：复制、粘贴、复制所有路径

### 复制文件路径格式
- 多文件路径使用 Windows 风格换行符 (`\r\n`) 分隔
- 路径列表末尾自动添加换行符
- 便于粘贴到编辑器或命令行
