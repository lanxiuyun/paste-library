## ⭐ 为什么选择 Paste Library

- 🏆 **零学习成本**：Alt+V 即用，无需改变习惯
- ⚡ **极速响应**：本地 SQLite，百万条记录秒开
- 🔒 **隐私优先**：数据本地存储，不上传云端
- 🎯 **智能搜索**：拼音、首字母、容错，打错字也能找到

## 🎯 核心功能

- 📋 **全格式通吃** - 代码片段、截图、文件路径，一个不漏
- 🔍 **模糊搜索** - 记得大概就行，"WL" 也能找到 "未来"
- 🏷️ **智能收藏** - 不用手动保存，打标签就是收藏
- 📋 **批量粘贴** - 一次选中多条，依次粘贴到目标应用
- ✏️ **即改即用** - 文本随时编辑，图片预览放大

### 📋 全格式通吃

- **文本**：纯文本、HTML 富文本、代码片段
- **图片**：截图、复制的图片，自动生成缩略图
- **文件**：文件路径、文件夹，一目了然

### 🔍 模糊搜索

- **拼音搜索**：打 "wenzi" 也能找到 "文字"
- **首字母搜索**：打 "WL" 找到 "未来"
- **容错搜索**：打错字也能智能匹配
- **标签搜索**：@标签快速分类检索

### ⌨️ 全局快捷键

- 默认 `Alt+V` 一键呼出
- 支持自定义快捷键
- 窗口自动隐藏，专注你的工作

### 🖱️ 灵活交互

- **单击/双击**：可配置为复制或直接粘贴
- **右键菜单**：复制、粘贴、纯文本、打开文件
- **拖拽**：支持跨应用拖拽发送
- **键盘导航**：↑↓ 选择，Enter 确认，ctrl+1-9 快捷粘贴

### 🏷️ 智能收藏

- 多标签分类管理
- 替代传统收藏，更灵活
- 快速筛选标签内容

### 📋 批量粘贴（待实现）

- 批量选中多条内容
- 一次粘贴，按序输出
- 购物车模式，轻松管理

### ✏️ 即改即用

- 文本内容随时编辑
- 图片预览放大
- 细节查看更方便
